Difference between "var", "let", and "const"

The main difference between is in their scope and assignability. Variables declared with "var" have the function-level scope and can be reassigned and redeclared, while similarity:"let" and "const" variables have the block-level scope, "let" variables can be updated but not re-declared, but "const" variables can neither be updated nor re-declared. Variables declared with const , just like let , are hoisted to the top of their global, local, or block scope – without a default initialization. var variables, are hoisted with a default value of undefined so they can be accessed before declaration without errors. similarity: var and let create variables that can be reassigned another value. const creates "constant" variables that cannot be reassigned another value var declarations are globally scoped or function scoped while similarity let and cons t are block scoped. While var and let can be declared without being initialized, const must be initialized during declaration.
