JavaScript was invented by Brendan Eich in 1995. It is Object-Based Language, needs to be integrated into an HTML program for execution. It was developed for Netscape 2, and became the ECMA-262 standard in 1997. After Netscape handed JavaScript over to ECMA, the Mozilla foundation continued to develop JavaScript for the Firefox browser. Mozilla's latest version was 1.8.5. (Identical to ES5). Internet Explorer (IE4) was the first browser to support ECMA-262 Edition 1 (ES1).

Different between Java and JavaScript


Java is a Programming language
Java is a pure Object Oriented Programming Language
Java is a Standalone language
Java programming language was developed by the " Sun Microsystems"
Java programming language, programs are saved with the ".java" extension
Java requires a large amount of memory
Java is stored on the host machine as the "Byte" code

JavaScript is a scripting language.
JavaScript is Object-Based Language.
JavaScript is not a standalone language, as it needs to be integrated into an "HTML program for execution".
JavaScript programming language was developed by the "Netscape."
JavaScript language programs are saved with the ".js" extension
JavaScript does not require that amount of memory
JavaScript is stored on the Host machine (client Machine) as the "source" text.

Difference between "var", "let", and "const"


The main difference between is in their scope and assignability. Variables declared with "var" have the function-level scope and can be reassigned
and redeclared, while
similarity:"let" and "const" variables have the block-level scope, "let" variables can be updated but not re-declared, but "const" variables can neither be updated nor re-declared.
Variables declared with const , just like let , are hoisted to the top of their global, local, or block scope – without a default initialization.
var variables, are hoisted with a default value of undefined so they can be accessed before declaration without errors. similarity: var and let create variables that can be reassigned another value. const creates "constant" variables that cannot be reassigned another value var declarations are globally scoped or function scoped while similarity let and const are block scoped. While var and let can be declared without being initialized, const must be initialized during declaration.
